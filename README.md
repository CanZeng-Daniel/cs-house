# 长沙买房攻略

![logo](./images/logo.jpg)
照片属于 © Carlos Chang


对于一部分人来说，房子是不动产，是财产，或是一笔钱。但是，对于另一部分人来说，房子，是爱，是家，是归宿。
所以，买房有时候也像爱情一样，需要点缘分，也需要点毅力，希望各位在摇号选房的路上不要失望，不要盲目，适合你的也许就在前方。
